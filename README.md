# Nextjs Introduction

```node
npx create-next-app@latest
or npx create-next-app@latest --typescript
```

```Endpoints
- /movies/:id : Get movie by :id
- /movies : List popular movies
- /movies/:id/videos : get videos for a movie by :id
```
